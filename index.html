Blang Programming Language

Blang is a modern, expressive programming language designed for simplicity and productivity. It combines familiar syntax with powerful features to create an enjoyable development experience.

Get Started

Installation

To install Blang, you need to clone the repository and build it from source. Follow these steps:

git clone https://github.com/blang-lang/blang.git
cd blang
npm install
npm link

This will make the blang command available globally on your system.

Usage

To use Blang, you can run Blang files or create new projects:

# Run a Blang file
blang run path/to/your/file.blang

# Create a new Blang project
blang create

# Create a new Blang project with default options
blang create -y

Syntax

Blang uses a clean, minimalist syntax inspired by popular programming languages. Here are some key syntactic elements:

Blang Equivalent Description
func function Defines a function
ret return Returns a value from a function
>> => Arrow function syntax
and && Logical AND operator
or || Logical OR operator
not ! Logical NOT operator

Variables

Variables in Blang are declared using the let keyword:

let name = "John"
let age = 30
let isStudent = true

Functions

Functions in Blang are defined using the func keyword:

func greet(name) {
    print("Hello, ${name}!")
}

greet("Alice")  // Outputs: Hello, Alice!

Arrow functions can be defined using the >> syntax:

let multiply = (a, b) >> {
    ret a * b
}

print(multiply(4, 5))  // Outputs: 20

Control Structures

Loops

loop (let i = 0; i < 5; i++) {
    print(i)

}

Modules

Blang uses a simple module system for code organization and reuse:

// In mathUtils.blang
func add(a, b) {
    ret a + b
}

export { add }

// In main.blang
import mathUtils from "mathUtils"

print(mathUtils.add(5, 3))  // Outputs: 8

String Interpolation

Blang supports string interpolation using ${} syntax:

let name = "Alice"
let age = 30
print("${name} is ${age} years old.")

Project Structure

When you create a new Blang project, it will have the following structure:

my-blang-project/
├── src/
│   └── main.blang
├── test/
├── package.json
└── README.md

The src/ directory contains your Blang source files, and the test/ directory is where you can place your test files.

Testing

Blang does not have a built-in testing framework yet. You can write your tests in separate .blang files and run them using the blang run command. For example:

// In test/math.test.blang
import { add } from "../src/mathUtils"

func testAdd() {
    let result = add(2, 3)
    if (result == 5) {
        print("Test passed: 2 + 3 = 5")
    } else {
        print("Test failed: expected 5, got ${result}")
    }
}

testAdd()

Run the test file using:

blang run test/math.test.blang

Examples

Factorial Calculator

func factorial(n) {
    if (n == 0 or n == 1) {
        ret 1
    } else {
        ret n * factorial(n - 1)
    }
}

print(factorial(5))  // Outputs: 120

Simple To-Do List

let todoList = []

func addTask(task) {
    todoList.push(task)
    print("Task added: ${task}")
}

func displayTasks() {
    print("To-Do List:")
    loop (let i = 0; i < todoList.length; i++) {
        print("${i + 1}. ${todoList[i]}")
    }
}

addTask("Buy groceries")
addTask("Write Blang code")
addTask("Exercise")

displayTasks()
